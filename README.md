# ReactJS Finances

Basic application created using ReactJS to manage personal finances (with RocketSeat)

## Required Configuration:

- [Node](https://nodejs.org)

## Installation:

```js
npm install
or
yarn
```

## Start:

```
npm start
or
yarn start
```
